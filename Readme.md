## Project Summary
1. Problem Formulation
2. Mixer Hamiltonian Formulation
3. Post-processing of mixer Hamiltonian
4. Error mitigation
5. Scale plot of $\beta$ and $\gamma$
6. QASM, noisy simulator and real harware benchmark
